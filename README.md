# Trade-OS BC

Trade-OS BC is an AI-powered communication and compliance system for trade businesses (towing, plumbing, pest control).

## Features
- SMS-based customer interaction (Twilio)
- AI-driven safety and compliance logic
- Admin dashboard (Streamlit)
- Backend API (FastAPI)
- Supabase database

## Tech Stack
- Python
- FastAPI
- Streamlit
- Supabase
- Twilio
